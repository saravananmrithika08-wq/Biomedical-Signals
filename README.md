# Biomedical Signals 📈

Analysis and visualisation of biomedical signals including ECG, EMG, 
and PPG using Python. Applying signal processing concepts from 
Biomedical Engineering coursework to real physiological data.

- **Student:** Mrithika S
- **Department:** Biomedical Engineering — 2nd Year
- **College:** Easwari Engineering College, Anna University

## 📁 Projects

| File | Signal | Description |
|------|--------|-------------|
| ecg_plotter.py | ECG | Plot and analyse ECG waveform |

## 🛠️ Libraries Used
- Python 3.x
- NumPy — numerical computation
- Matplotlib — signal visualisation
- SciPy — signal filtering and processing

## 📊 Signals Covered
- **ECG** — Electrocardiogram (heart electrical activity)
- **EMG** — Electromyogram (muscle electrical activity)
- **PPG** — Photoplethysmogram (pulse/blood volume)

## 📌 Dataset Sources
- PhysioNet — free biomedical signal database
- Simulated signals using NumPy
